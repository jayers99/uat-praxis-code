# Hello World CLI

A minimal greeting CLI demonstrating Praxis governance.

## Usage

```bash
# Default greeting
hello-world
# Output: Hello, World!

# Custom greeting
hello-world --name Praxis
# Output: Hello, Praxis!
```

## Development

```bash
poetry install
poetry run pytest
poetry run hello-world
```

This project demonstrates the full Praxis lifecycle from Capture to Sustain. See `docs/` for stage artifacts.
